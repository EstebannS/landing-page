# 🌐 Landing Page Personal

Página web personal responsiva construida con **HTML5, CSS3 y JavaScript vanilla** — sin frameworks ni dependencias.

![Preview](preview.png)

## 🚀 Ver en vivo

**[TU_USUARIO.github.io/01-landing-page](https://TU_USUARIO.github.io/01-landing-page)**

## ✨ Características

- Diseño oscuro moderno con tipografía editorial
- 100% responsive (mobile, tablet, desktop)
- Animaciones CSS y JavaScript con Intersection Observer
- Contadores animados al hacer scroll
- Barras de skill con animación de entrada
- Navegación sticky con efecto blur al hacer scroll
- Menú hamburguesa para móvil
- Formulario de contacto con validación
- Botón "volver arriba"
- Sin dependencias externas (solo Google Fonts)

## 🛠️ Tecnologías

| Tecnología | Uso |
|------------|-----|
| HTML5 semántico | Estructura y accesibilidad |
| CSS3 (Variables, Flexbox, Grid) | Diseño y animaciones |
| JavaScript ES6+ | Interactividad y observers |
| GitHub Pages | Deploy gratuito |

## 🌍 Deploy con GitHub Pages

1. Ve a tu repositorio en GitHub
2. **Settings** → **Pages**
3. Source: **Deploy from a branch** → `main` → `/ (root)`
4. Espera 1–2 minutos y tu sitio estará en:
   `https://TU_USUARIO.github.io/01-landing-page`
